# Guesthouse Landing Page

Welcome to the Guesthouse Landing Page!

## Overview
This landing page is designed to provide visitors with all the essential information about our guesthouse, making it easy to book a stay and learn about the amenities we offer.

## Features
- **User-friendly Interface:** An easy-to-navigate layout that directs users to key information.
- **Image Gallery:** A visually appealing gallery showcasing the guesthouse's interiors and surroundings.
- **Booking Options:** Convenient booking forms allowing users to reserve rooms instantly.
- **Contact Information:** Clear display of contact details for queries and assistance.

## Technologies Used
- HTML
- CSS
- JavaScript
